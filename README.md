## 🚀 デプロイ方法

### 方法1: Publishタブを使用（推奨）
1. Publishタブをクリック
2. 「Deploy」ボタンをクリック
3. 自動的に公開URLが生成されます

### 方法2: 外部サービスを使用

#### Netlify
1. [Netlify](https://www.netlify.com/)にアクセス
2. 「Add new site」→「Deploy manually」
3. すべてのファイルをドラッグ&ドロップ
4. 自動的にデプロイされます

#### Vercel
1. [Vercel](https://vercel.com/)にアクセス
2. 「New Project」をクリック
3. GitHubリポジトリをインポートまたは手動アップロード
4. デプロイ完了

## 💡 トラブルシューティング

### フォームが送信できない
1. Formspree IDが正しく設定されているか確認
2. ブラウザのコンソールでエラーを確認
3. メールアドレスが正しいか確認

### デジタルエフェクトが表示されない
1. JavaScriptが有効になっているか確認
2. ブラウザのバージョンが最新か確認
3. Canvas APIに対応しているか確認

### モバイルでレイアウトが崩れる
1. ビューポートメタタグが設定されているか確認
2. ブラウザのキャッシュをクリア
3. CSSのメディアクエリを確認
